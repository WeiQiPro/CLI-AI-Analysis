CLI Tool for AI analysis.

Using Deno it will parse an sgf, send game to KataGo, write KataGo Queries to the SGF.
CLI tool is capable of doing batch analysis.

Finished:
- Parse SGFS
- Open KataGo
- Read and write Queries.
